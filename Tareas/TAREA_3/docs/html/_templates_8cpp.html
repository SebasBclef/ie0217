Returns
Matriz resultante de la operación de multiplicación.
Exceptions
std::invalid_argumentSi las matrices no cumplen las condiciones para la multiplicación.

Este operador permite multiplicar dos matrices del mismo tipo. Antes de realizar la multiplicación, se verifica si las matrices cumplen las condiciones necesarias para esta operación. En caso contrario, se lanza una excepción de tipo std::invalid_argument.

Note
La operación de multiplicación se realiza utilizando el método multiplicacion de la clase OperacionesBasicas.
