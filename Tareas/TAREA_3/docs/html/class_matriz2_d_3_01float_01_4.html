Detailed Description

Clase especializada para matrices de números flotantes.

Esta clase template está especializada para manejar matrices de números flotantes. Almacena la matriz en un contenedor de vector bidimensional y proporciona métodos para operaciones y manipulación de la matriz.

Template Parameters
TTipo de dato de los elementos de la matriz (en este caso, float).

void Matriz2D< float >::ingresarDatos ( )
inline

Permite al usuario ingresar datos para llenar la matriz de flotantes.

Este método solicita al usuario que ingrese los datos para llenar una matriz de números flotantes. Utiliza un bucle anidado para recorrer cada posición de la matriz y solicita el valor flotante de cada elemento al usuario. Verifica si el valor ingresado es un número flotante y maneja posibles errores de entrada.

Note
Este método asume que la matriz ya ha sido inicializada con las dimensiones adecuadas.
Note
El bucle externo recorre las filas de la matriz.
El bucle interno recorre las columnas de la matriz.
Se muestra un mensaje solicitando el valor flotante del elemento en la posición (i, j).
Se verifica si el valor ingresado es un número flotante. En caso de error, se muestra un mensaje de error, se limpia el buffer de entrada y se solicita nuevamente el ingreso del valor.
Se establece el valor ingresado en la posición (i, j) de la matriz.
