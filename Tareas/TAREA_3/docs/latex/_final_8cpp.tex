Función principal del programa para realizar operaciones con matrices. 

Se solicita al usuario ingresar el número de filas y columnas para las matrices. Luego, se crea una matriz de enteros, una matriz de flotantes y una matriz de números complejos según las preferencias del usuario. Se realizan operaciones de suma, resta o multiplicación entre matrices, y se muestra el resultado.

\begin{DoxyReturn}{Returns}
0 si el programa se ejecuta correctamente. 
\end{DoxyReturn}
Matriz de enteros 1.

Se crea una matriz de enteros con el número de filas y columnas especificado por el usuario.

Matriz de enteros 2.

Se crea otra matriz de enteros con el mismo número de filas y columnas que la Matriz de enteros 1.

Matriz de flotantes 1.

Se crea una matriz de flotantes con el mismo tamaño que las matrices de enteros.

Matriz de números complejos 1.

Se crea una matriz de números complejos con el mismo tamaño que las matrices de enteros.

Solicitar al usuario el tipo de matriz para la Matriz 1.

Se presenta un mensaje en consola pidiendo al usuario que seleccione el tipo de matriz para la Matriz 1. Las opciones son\+:
\begin{DoxyItemize}
\item 1\+: Matriz de enteros.
\item 2\+: Matriz de flotantes.
\item 3\+: Matriz de números complejos.
\end{DoxyItemize}

\begin{DoxyReturn}{Returns}
La opción seleccionada por el usuario.
\end{DoxyReturn}
Crear la Matriz 1 según la opción seleccionada por el usuario.

Se utiliza una estructura de switch para determinar el tipo de matriz a crear\+:
\begin{DoxyItemize}
\item Si la opción es 1, se crea y muestra una matriz de enteros.
\item Si la opción es 2, se crea y muestra una matriz de flotantes.
\item Si la opción es 3, se crea y muestra una matriz de números complejos.
\item En caso de una opción no válida, se muestra un mensaje de error y se finaliza el programa.
\end{DoxyItemize}

Solicitar al usuario que ingrese el número de filas y columnas, y seleccione el tipo de matriz para la Matriz 2.

Se solicita al usuario que ingrese el número de filas y columnas para la Matriz 2, y luego seleccione el tipo de matriz\+:
